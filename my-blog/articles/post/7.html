安装教程：

运行后端：

cd backend
echo "OPENAI_API_KEY=sk-your-key" > .env
poetry install
poetry shell
poetry run uvicorn main:app --reload --port 7001

如果您想使用 Anthropic，请将与您的 Anthropic 的 API 密钥一起添加到 backend/.env 文件中。

运行前端：

cd frontend
yarn
yarn dev

打开 http://localhost:5173 以使用该应用程序。

调试模式：

如果您不想浪费 GPT4-Vision 积分，您可以在模拟模式下运行后端（该模式会传输预先录制的响应）：

MOCK=true poetry run uvicorn main:app --reload --port 7001

Docker 一键安装命令：

echo "OPENAI_API_KEY=sk-your-key" > .env
docker-compose up -d --build
